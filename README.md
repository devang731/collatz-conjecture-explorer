# Computational Exploration of the Collatz Conjecture

## What is this?
The Collatz conjecture states: take any positive integer n.
If even, divide by 2. If odd, multiply by 3 and add 1.
Repeat. The conjecture says you always reach 1.
Paul Erdos said "Mathematics is not yet ready for such problems."
This project is my attempt to go as deep as possible anyway.

## What I am doing
- Testing the conjecture across billions of integers in C++
- Visualising stopping-time distributions
- Searching systematically for anomalies in prime inputs
- Documenting every finding and every failure publicly

## Why
Because I could not stop thinking about why such a simple rule
produces behaviour that mathematics cannot yet predict.
